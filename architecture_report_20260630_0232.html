Design Principles Applied

PrincipleImplementation
Defence in DepthThree-tier isolation — web, app, data. Traffic must traverse layers sequentially
Least PrivilegeEach NSG only allows minimum required ports from minimum required sources
Zero TrustNo implicit trust between subnets — all cross-tier traffic explicitly controlled by NSG rules
Infrastructure as CodeEntire architecture defined in Bicep — version controlled, repeatable, consistent
DDoS ResilienceAzure DDoS Protection Standard enabled at VNet level — automatic mitigation
ParameterisationEnvironment name and IP ranges parameterised — same template deploys to dev/staging/prod
